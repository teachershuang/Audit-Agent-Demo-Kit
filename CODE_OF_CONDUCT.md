# 行为准则

为确保社区协作环境清晰、专业、可持续，参与本项目的所有人都应遵守以下准则：

## 我们鼓励

- 基于事实和复现步骤讨论问题
- 对代码、设计和架构进行明确、具体的反馈
- 尊重不同背景和经验水平的贡献者
- 在提交问题时提供日志、截图或最小复现

## 我们不接受

- 人身攻击、辱骂、歧视或骚扰
- 恶意刷屏、引战或无关争论
- 泄露他人隐私、企业内部数据或未授权资料
- 提交含敏感密钥、真实账号或生产数据的内容

## 处理原则

维护者有权删除不合适内容、关闭违规讨论，或限制严重违规账号的互动权限。

## 安全与隐私

请默认使用脱敏样例，不要在 Issue、PR、日志或截图中提交真实合同、企业数据、个人信息或密钥。
